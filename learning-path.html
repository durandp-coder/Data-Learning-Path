DataFrames, series, merges, groupby, pivot tables, limpieza de datos. Es básicamente Excel/SQL pero con superpoderes en Python.

▶Keith Galli — Pandas Complete Intro (YouTube)YTGratis
◉Kaggle — Pandas (certificado gratuito)CertGratis

semanas
Visualización con Python

Matplotlib, Seaborn y Plotly. Visualización programática que complementa tu conocimiento de Tableau y Power BI.

▶freeCodeCamp — Matplotlib & Seaborn (YouTube)YTGratis
◉Kaggle — Data Visualization (certificado)CertGratis
2
semanas
Análisis Exploratorio de Ventas con Python

Descarga un dataset de ventas de Kaggle (ej. Superstore o Brazilian E-Commerce). Limpia los datos con Pandas, calcula KPIs como revenue por categoría, tasa de retorno y ticket promedio. Genera un reporte visual con Seaborn/Plotly. El objetivo es replicar en Python algo que ya sabes hacer en Excel o Tableau.

Python Pandas Seaborn EDA ~1 semana
2
Mes 3 Gratis
SQL Intermedio — Fundamento Sólido
JOINs complejos, subqueries, agregaciones, modelado básico
+
JOINs Avanzados & Subqueries

INNER, LEFT, RIGHT, FULL OUTER, CROSS JOIN y SELF JOIN con casos reales. Subqueries correlacionadas vs no correlacionadas. EXISTS vs IN. Cuándo cada uno es la herramienta correcta.

▶techTFQ — SQL JOINs explicados a fondo (YouTube)YTGratis
◉SQLZoo — Práctica interactiva de JOINsGratis
1
semana
Agregaciones & Agrupaciones

GROUP BY con HAVING, ROLLUP, CUBE y GROUPING SETS. Diferencias clave entre WHERE y HAVING. Funciones de agregación: COUNT DISTINCT, PERCENTILE_CONT, MODE, MEDIAN.

▶Ankit Bansal — GROUP BY avanzado (YouTube)YTGratis
◉Mode Analytics — Aggregate FunctionsGratis
1
semana
CTEs & Queries Complejas

Common Table Expressions (WITH clause) para organizar lógica compleja. CTEs múltiples encadenadas. Diferencia entre CTE vs subquery vs vista temporal. Cuándo usar cada uno.

▶techTFQ — CTEs en SQL (YouTube)YTGratis
◉Mode Analytics — CTEs TutorialGratis
1
semana
Modelado de Datos Básico

Star schema vs snowflake schema. Tablas de hechos y dimensiones. Normalización y desnormalización. Diseñar un modelo simple de datos para análisis. Fundamento para entender dbt y Databricks.

▶Kahan Data Solutions — Star Schema explicado (YouTube)YTGratis
▶Alex The Analyst — Data Modeling para analistasYTGratis
1
semanas
Dashboard de Negocio con SQL Puro

Usa un dataset público (ej. Northwind o AdventureWorks). Modela un star schema simple con tablas de hechos y dimensiones. Escribe queries con JOINs, CTEs y agregaciones para responder 5 preguntas de negocio reales: ¿Cuál es el producto más rentable? ¿Cuál es el cliente con mayor LTV? ¿Cuál es el mes con mayor crecimiento YoY? Exporta los resultados a Tableau/Power BI.

SQL CTEs JOINs Star Schema ~1 semana
3
Meses 4–5 Gratis + Pago
SQL Avanzado & Databricks
Window functions, optimización, Spark SQL y la plataforma Databricks
+
Window Functions

ROW_NUMBER, RANK, DENSE_RANK, NTILE. LAG y LEAD para comparaciones temporales. FIRST_VALUE, LAST_VALUE, NTH_VALUE. OVER() con PARTITION BY y ORDER BY. El tema más pedido en entrevistas técnicas de datos.

▶Ankit Bansal — Window Functions Masterclass (YouTube)YTGratis
◉LeetCode — Top SQL 50 (práctica real)Gratis
▶techTFQ — SQL Interview Queries (YouTube)YTGratis
2
semanas
Optimización de Queries

Query execution plans, índices, particionamiento, estadísticas del optimizador. Cómo diagnosticar queries lentas. EXPLAIN ANALYZE. Diferencias clave al optimizar en bases de datos columnar (como Databricks) vs transaccional.

▶Database Star — SQL Query Optimization (YouTube)YTGratis
◉Use The Index, Luke — Guía de índices SQLGratis
1
semana
Fundamentos de Databricks

La plataforma, Notebooks, Clusters, Unity Catalog, Delta Lake. Databricks SQL para analistas. Certificación oficial gratuita de Databricks.

◉Databricks — Lakehouse Fundamentals (cert. gratis)CertGratis
▶Adam Marczak — Databricks Tutorial (YouTube)YTGratis
◉Databricks Community Edition (gratuita para practicar)Gratis
2
semanas
Apache Spark & PySpark

Procesamiento distribuido, DataFrames de Spark, transformaciones lazy, Spark SQL. El motor detrás de Databricks. Cómo escalar código Python a millones de registros.

▶Codebasics — PySpark Tutorial Completo (YouTube)YTGratis
◉Coursera — Intro Big Data with Spark (IBM, audit gratis)Gratis
2
semanas
Análisis de Cohortes con Window Functions en Databricks

Sube un dataset de transacciones a Databricks Community Edition. Usa window functions para calcular retención de cohortes mes a mes (un análisis clásico de e-commerce). Identifica el mes de primera compra de cada usuario con FIRST_VALUE, luego calcula cuántos regresan en meses posteriores con LAG/LEAD. Visualiza la matriz de cohortes en un notebook de Databricks.

Window Functions Databricks Cohort Analysis Spark SQL ~1.5 semanas
4
Meses 6–8 Inversión recomendada
Data Engineering Moderno
dbt, Delta Lake, pipelines y certificación Databricks
+
dbt (Data Build Tool)

Transformaciones SQL modulares, tests automáticos, documentación, linaje de datos. La herramienta más adoptada en el stack moderno de datos. Se integra directamente con Databricks.

◉dbt Labs — dbt Fundamentals (certificado gratuito)CertGratis
▶Jayzern — dbt Tutorial Completo (YouTube)YTGratis
3
semanas
Delta Lake & Data Lakehouse

ACID transactions en data lakes, versioning, time travel, merge operations (UPSERT). La base de la arquitectura Lakehouse. Medallion architecture: Bronze → Silver → Gold.

◉Databricks — Delta Lake Fundamentals (gratis)Gratis
▶Advancing Analytics — Delta Lake Deep DiveYTGratis
2
semanas
Git & GitHub para Datos

Control de versiones para proyectos de datos. Pull requests, branches, .gitignore para notebooks. GitHub Actions básico. Cómo versionar modelos dbt y notebooks de Databricks.

▶freeCodeCamp — Git & GitHub (YouTube)YTGratis
1
semana
Certificación Databricks Associate

Databricks Certified Data Engineer Associate. Reconocida en el mercado, valida tu conocimiento de Spark, Delta Lake y pipelines. La inversión principal de todo el learning path.

◉Databricks — Data Engineer Associate Exam (~$200)$200Cert
▶Prep: Databricks Associate Study Guide (YouTube)YTGratis
3
semanas
Pipeline Medallion con dbt + Delta Lake

Construye un pipeline completo en Databricks Community Edition usando la arquitectura Medallion. Bronze: ingesta datos raw de un CSV público (ej. NYC Taxi o ventas). Silver: limpieza y transformaciones con PySpark + dbt. Gold: métricas agregadas listas para análisis. Versiona todo en GitHub. El pipeline debe ser reproducible con un solo comando.

dbt Delta Lake PySpark Medallion GitHub ~2 semanas
5
Meses 9–10 Gratis + Pago
AI Aplicada al Análisis de Datos
ML básico, LLMs para análisis, Databricks AI & MLflow
+
Machine Learning para Analistas

Regresión, clasificación, clustering aplicados a datos de negocio con scikit-learn. Sin matemáticas profundas: enfoque en cuándo usar cada modelo y cómo interpretar resultados para stakeholders.

◉Kaggle — Intro to ML (certificado gratis)CertGratis
◉Kaggle — Intermediate ML (certificado gratis)CertGratis
▶Patrick Loeber — ML con scikit-learn (YouTube)YTGratis

semanas
LLMs & AI para Análisis de Datos

Usar LLMs (GPT, Claude) para generar SQL/Python, analizar datasets con lenguaje natural, crear reportes automáticos y construir agentes de análisis. Prompt engineering para analistas de datos.

◉DeepLearning.AI — Prompt Engineering (gratis)Gratis
◉DeepLearning.AI — LangChain (gratis)Gratis
▶Alejandro AO — AI Data Analyst Agent (YouTube)YTGratis

semanas
Databricks AI & MLflow

Databricks AutoML, Feature Store, MLflow para tracking de experimentos y reproducibilidad, Model Registry. Cómo integrar modelos de ML dentro de pipelines de datos en Databricks.

◉Databricks — ML Foundations (gratis)Gratis
▶Databricks — MLflow Tutorial (YouTube)YTGratis
2
semanas
Agente de Análisis con LLM + Modelo Predictivo

Combina un modelo de ML (ej. predicción de churn o forecast de demanda) con un LLM que interprete los resultados en lenguaje natural. Registra el experimento en MLflow. El agente debe poder responder preguntas como "¿Qué clientes tienen mayor riesgo de abandono este mes?" consultando los datos de Databricks y generando una explicación automática.

scikit-learn MLflow LangChain Databricks ~2 semanas
